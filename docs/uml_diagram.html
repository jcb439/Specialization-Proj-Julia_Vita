UML Diagram

Understanding the units within our system

This UML diagram illustrates the key components, relationships, and data flows within the Shopping Speed Bump system. It shows how different classes interact to provide a seamless intervention experience during online shopping.

Key components (hover to reveal details):

User & Shopping Context
User profiles, preferences, and current shopping session management
Item & Wardrobe Management
Product details, wardrobe inventory, and similarity detection algorithms
Category & Comparison
Classification system and visual comparison tools
Intervention Logic
Intensity settings and trigger mechanisms for different intervention levels
Reflection Prompt →
Relationships between the user and their set reflection prompt intensity in settings
Wardrobe Query →
Comparisons between shopping data and the user's existing closet inventory

Tip: Use both horizontal and vertical scrolling to explore the full diagram. Use zoom controls to focus on specific class relationships and details.
